# SpaceTrader

Overview

SpaceTrader is a C++-based trading simulation game developed as part of an Object-Oriented Programming course project. Players manage resources, trade goods, earn credits, and make strategic decisions while exploring a space-themed environment.

# Features
User-friendly graphical interface using SFML
Trading system for buying and selling resources
Player inventory management
Currency and profit tracking
Multiple game locations/markets
Save and load game data
Interactive gameplay mechanics
OOP Concepts Used

This project demonstrates the following Object-Oriented Programming concepts:

# Encapsulation
Data members are protected within classes and accessed through member functions.
# Inheritance
Derived classes extend functionality from base classes.
# Polymorphism
Virtual functions are used to provide flexible behavior.
# Abstraction
Complex game systems are simplified through class design.
